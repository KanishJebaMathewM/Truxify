# Issue #4506

## Description
Added support for a "System" dark mode setting in the Driver App in addition to the existing "Light" and "Dark" toggles. Also localized the theme selector labels so they dynamically adapt to the user's selected language.

## Changes
- **apps/driver/lib/screens/profile_screen.dart**: Updated the `SegmentedButton` in `_ThemeModeTile` to include a `ThemeMode.system` option.
- **apps/driver/lib/screens/profile_screen.dart**: Replaced hardcoded strings ("Theme", "Light", "Dark") with `AppLocalizations.of(context)`.
- **apps/driver/lib/l10n/app_en.arb**: Added strings for `theme`, `system`, `light`, and `dark`.

## Testing
1. Launch the Driver app.
2. Navigate to the Profile screen.
3. Observe the Theme toggle now has three segments: System, Light, Dark.
4. Toggle between the modes to verify the theme updates instantly and respects system settings when "System" is selected.
